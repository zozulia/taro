Структура курса

Система координат в эзотерике

Вы узнаете, в каких эзотерических направлениях работают карты Таро.

"Красная книжечка" таролога

Только для практикующих тарологов - правила работы с картами Таро при консультировании клиентов.

Исключительно проверенные значения

В процессе обучения Вы получаете только те значения карт, которые проверены на практике профессиональных тарологов годами.

Объемные толкования

Общие значения каждой из 78 карт (по 3 значения на каждую карту) + значения в отношениях + значения в бизнесе.

Только работающие расклады

Четкие схемы исключительно надежных раскладов, которые не подведут даже в самых сложных вопросах.

Практика, практика и ещё раз практика

Включение практических блоков по желанию группы + у каждого ученика на завершающем занятии будет возможность выйти в эфир, самостоятельно сделать расклад с толкованием и получить обратную связь от преподавателя.
